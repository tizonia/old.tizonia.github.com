Tizonia Download Tool

For now, the recommemded way to try Tizonia is to clone the repos and build everything from source. The repos are mirrored on Github and Bitbucket.

However, the manual installation is tedious. Instead, you can use Tizonia's download tool, which will do everything for you, including downloading the Creative Commons-licensed media files that are used throughout the Tizonia unit tests and Skema's test suites.

Clone the 'Tizonia Download Tool' repo

      # From GitHub, clone the following repo...
      $ git clone https://github.com/tizonia/tizonia-download-tool.git

Run the tool

      # Change dir to the 'bin' directory
      $ cd tizonia-download-tool/bin
      #
      # Run the tool without options for bring up the help
      #
      $ ./tizonia-download-tool
      Need to specify a target directory for non-interactive runs.
      Usage: tizonia-download-tool [options]

      Options:
        --version             show program's version number and exit
        -h, --help            show this help message and exit
        -i, --interactive     use this option to enable an interactive run
        -t TARGET_DIR, --target-dir=TARGET_DIR
                              TARGET_DIR represents the base directory that will be
                              used by the tool to: a) download the repos and test
                              files, b) install the software (i.e., the lib/
                              include/ share/ and bin/ subdirs). This option is
                              required when --interactive or -i option is not
                              specified
      #
      # e.g. execute the tool like this
      #
      $ ./tizonia-download-tool -i -t $HOME/temp/tizonia
      #
