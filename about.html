About Tizonia

Tizonia is a multimedia framework for Linux based on OpenMAX IL 1.2.

The project currently includes a number of libraries written in C or C++, including an OpenMAX IL base component library, an implementation of the OpenMAX IL Core, and OS abstraction/utility library, a resource management framework and a growing number of OpenMAX IL 1.2 components.

Tizonia is released under the GNU Lesser General Public License version 3 license.

About Skema, Tizonia's test execution framework

Skema is a test execution framework written in Python that can be used to create arbitrary multimedia graphs and use cases to exercise the Tizonia OpenMAX IL components.

Skema borrows ideas and some of its code from Linaro's excellent Lava-test framework.

Skema is released under the GNU General Public License version 3 license.
