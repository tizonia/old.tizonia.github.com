Tizonia OpenMAX IL

An open source implementation of OpenMAX IL 1.2

Tizonia is an experimental implementation of the OpenMAX IL 1.2 provisional specification from The Khronos Group.

Tizonia OpenMAX IL consists of a number of resources:

  • A fairly complete implementation of a base OpenMAX IL component library (supports Base and Interop profiles)
  • An IL Core implementation.
  • An OS abstraction/utility library, including wrappers for memory, threading and synchronization primitives, FIFO and priority queues, dynamic arrays, config file parsing, uuids, small object allocation, etc
  • A Resource Management (RM) framework, including a C client library, and a D-Bus-based RM server written in C++.
  • A number of sample IL components, including an mp3 decoder (based on libmad), a vp8 decoder (based on libvpx), a SDL-based YUV renderer, etc.

There is also a a test execution framework written in Python called Skema, that allows the definition and execution of arbitrary OpenMAX IL graphs and use cases using an easy-to-program XML syntax. Skema internally uses Python's ctypes to dynamically load and use the OpenMAX IL Core. Skema can be easily extended to support new functionality and test scenarios by defining additional XML tags also written in Python.
