Tizonia OpenMAX IL

An experimental implementation of the OpenMAX IL 1.2 provisional specification

About the Tizonia project.

Tizonia is an experimental implementation of the OpenMAX IL 1.2 provisional specification by The Khronos Group.

The project includes a number of modules written in C or C++, including a base OpenMAX IL component library, an implementation of the OpenMAX IL Core, and OSAL/utility library, a resource management framework and a number of IL components, like an MP3 decoder, a VP8 decoder, and audio and video renderers.

There is also a test execution framework written in Python that can be used to create arbitrary multimedia graphs and use cases that exercise the Tizonia OpenMAX IL components.
