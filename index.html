Tizonia OpenMAX IL

An open source implementation of OpenMAX IL 1.2

Tizonia is an experimental implementation of the OpenMAX IL 1.2 provisional specification from The Khronos Group.

The project currently includes a number of libraries written in C or C++, including an OpenMAX IL base component library, an implementation of the OpenMAX IL Core, and OS abstraction/utility library, a resource management framework and a number of OpenMAX IL components, like an MP3 decoder, a VP8 decoder, and audio and video renderers.

There is also a test execution framework written in Python that can be used to create arbitrary multimedia graphs and use cases to exercise the Tizonia OpenMAX IL components.
